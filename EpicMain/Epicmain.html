Service Desk Epic Support Plan

Epic is an EMR application used in the CHI Enterprise and uses "LDAP" authentication, meaning the user's 'Active Directory' account/password is used for logging in.

Important: EpicCare Link (ECL) is not an AD linked application therefore these users will not be in Service Management. If you cannot find an existing user record and the person is a non-employee user of Epic via EpicCare Link, tag the caller as "EpicCare Caller".

(Username example: ECLMLEGGETT, AECLMLEGGETT (A=site admin), PECLMLEGGETT (P=payer) or APECLMLEGGETT (A=site admin, P=payer))
